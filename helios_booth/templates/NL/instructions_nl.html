Beste kiezer,
Om succesvol je stem uit te brengen moeten onderstaande stappen doorlopen worden.
De eerste 4 stappen (figuur: de licht-grijze blokken) zijn pagina's lokaal in je browser. Dit wil zeggen dat er geen verbinding is met de server en er dus ook geen (geheime) informatie kan doorgespeeld worden:

  1. Lees deze instructies aandachtig.

  2. Kies je stem.
    Je kan eenvoudig terug en verder navigeren.

  3. Kijk je keuzes na in het overzicht.
    Je keuzes worden veilig geëncrypteerd binnen je browser om ervoor te zorgen dat je stem geheim blijft.

  4. Verifieer de enryptie (optioneel).
    Om er zeker van te zijn dat je stem juist is geëncrypteerd kan je ervoor kiezen om deze te verifiëren. Je kan dit proces zo vaak doorlopen als je wilt, tot je er van overtuigd bent dat het stemsysteem correct werkt.
Na de encryptie en de optionele verificatie zal je moeten inloggen om je stem te kunnen indienen (figuur: donkere blokken). Vanaf hier zal dus verbindingen gemaakt worden met de Helios-server. Je stem is echter al geëncrypteerd en blijft dus geheim.
  1. Dien je geëncrypteerde keuzes in.
    Je zal gevraagd worden om in te loggen en je krijgt een verificatie code. Het is belangrijk dat je deze code bewaard zodat je later kan nakijken of je stem daadwerkelijk is meegeteld. Wanneer je stem correct is ingediend zal je een confirmatie te zien krijgen.

Check je verificatie code.
Op het einde van de stemming wordt een lijst met verificatie codes die geteld zijn gepubliceerd. Als je er zeker van wilt zijn dat je stem is meegeteld kan je jouw verificatie code opzoeken in de lijst.
